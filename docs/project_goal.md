# project_goal

## 프로젝트 한 줄 설명
캘린더에서 오늘 운동 루틴을 미리 정하고, 워치에서 간단하게 실행·확인할 수 있는 근력운동 관리 앱

## 이 프로젝트의 핵심 목표
- 사용자가 폰에서 날짜별 루틴을 빠르게 만들 수 있어야 한다
- 사용자가 운동 중 워치만 보고 현재 세트와 휴식 흐름을 이해할 수 있어야 한다
- 운동 후 캘린더에서 그날의 운동 기록을 자연스럽게 돌아볼 수 있어야 한다

## 주요 사용자
- 근력운동을 꾸준히 하는 개인 사용자
- 운동 전 루틴 결정을 미리 끝내고 싶은 사용자
- 운동 중 휴대폰 확인을 줄이고 싶은 사용자

## 사용자가 최종적으로 할 수 있어야 하는 것
- 특정 날짜에 운동 루틴을 저장한다
- 워치에서 오늘 루틴을 실행한다
- 운동 완료 후 기록을 캘린더에서 확인한다

## 지금 단계에서 가장 중요한 가치
복잡한 분석보다, 운동 계획 → 실행 → 회고 흐름을 끊기지 않게 만드는 것
