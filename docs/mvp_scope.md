# mvp_scope

## 이번 MVP에서 포함할 것
- 월간 캘린더에서 날짜 선택
- 날짜별 운동 루틴 생성 및 수정
- 운동 이름, 세트 수, 반복 수, 무게, 휴식 시간 입력
- 워치에서 오늘 루틴 시작
- 현재 운동 / 세트 / 반복 수 / 무게 / 휴식 타이머 표시
- 세트 완료 및 다음 세트 진행
- 운동 완료 후 날짜별 기록 저장
- 캘린더에서 완료 여부와 날짜 상세 기록 조회

## 이번 MVP에서 제외할 것
- 심박수 기반 휴식 안내
- 고급 통계 그래프
- 세밀한 퍼포먼스 분석
- 커뮤니티 / 소셜 기능
- 과도하게 복잡한 기록 옵션
- 자동 추천 / AI 코칭

## 이번 MVP의 완료 기준
- 사용자가 폰에서 특정 날짜의 루틴을 만들 수 있다
- 사용자가 워치에서 오늘 루틴을 시작하고 세트를 진행할 수 있다
- 운동 완료 후 캘린더에 기록이 남는다
- 핵심 흐름이 큰 오류 없이 1회 끝까지 동작한다

## 작업 원칙
- 한 번에 작업 1개만 진행
- 작은 단위로 검증
- 큰 구조 변경은 보수적으로
- UI보다 핵심 흐름 완성을 우선한다
